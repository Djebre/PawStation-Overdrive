# À Propos de PawStation Overdrive

## Bienvenue sur PawStation Overdrive !

PawStation Overdrive est une collection de jeux arcade sur le thème "Space Groove", conçue spécialement pour les conventions furry. Plongez dans un univers disco cosmique où les réflexes et l'adrénaline se mêlent aux néons et à la musique électronique.

## Notre Mission

Offrir une expérience de jeu unique et mémorable lors des conventions, avec des mécaniques simples mais addictives, et un leaderboard compétitif pour défier vos amis !

## Les Jeux

### Groove Orbit Runner (Disponible)
Un jeu d'arcade skill-based où vous tournez autour d'une planète disco en évitant des obstacles. Changez d'orbite au bon moment et survivez le plus longtemps possible !

### Space Groove Drift (Bientôt disponible)
Une expérience chill où vous surfez sur des ondes cosmiques dans un univers vaporwave relaxant.

### Groove Arena Overdrive (Bientôt disponible)
Un jeu hardcore où vous devez dasher au rythme de la musique pour survivre dans une arène remplie d'ennemis dansants.

## Technologies Utilisées

**Frontend:**
- React 19 - Framework UI moderne
- Phaser 3.90 - Moteur de jeu HTML5
- Tailwind CSS - Design system
- React Router - Navigation
- Lucide React - Icônes

**Backend:**
- FastAPI - Framework Python haute performance
- MongoDB - Base de données NoSQL
- Pydantic - Validation des données
- Motor - Driver MongoDB asynchrone

**DevOps:**
- Docker & Docker Compose - Conteneurisation
- Nginx - Reverse proxy et serveur web
- Let's Encrypt - Certificats SSL
