# À Propos de PawStation Overdrive

## Bienvenue sur PawStation Overdrive !

PawStation Overdrive est une collection de jeux arcade sur le thème "Space Groove". Plongez dans un univers disco cosmique où les réflexes et l'adrénaline se mêlent aux néons et à la musique électronique.

## Technologies Utilisées

**Frontend:**
- React 19 - Framework UI moderne
- Phaser 3.90 - Moteur de jeu HTML5
- Tailwind CSS - Design system
- React Router - Navigation
- Lucide React - Icônes

**Backend:**
- FastAPI - Framework Python haute performance
- MongoDB - Base de données NoSQL
- Pydantic - Validation des données
- Motor - Driver MongoDB asynchrone

**DevOps:**
- Docker & Docker Compose - Conteneurisation
- Nginx - Reverse proxy et serveur web
- Let's Encrypt - Certificats SSL
