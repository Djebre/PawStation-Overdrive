# À Propos de PawStation Overdrive

## Bienvenue sur PawStation Overdrive !

PawStation Overdrive est une collection de jeux arcade sur le thème "Space Groove". Plongez dans un univers disco cosmique où les réflexes et l'adrénaline se mêlent aux néons et à la musique électronique.

## Technologies Utilisées

**Frontend:**
- React 19 - Framework UI moderne
- Phaser 3.90 - Moteur de jeu HTML5
- Tailwind CSS - Design system
- React Router - Navigation
- Lucide React - Icônes

**Backend:**
- FastAPI - Framework Python haute performance
- MongoDB - Base de données NoSQL
- Pydantic - Validation des données
- Motor - Driver MongoDB asynchrone

**DevOps:**
- Docker & Docker Compose - Conteneurisation
- Nginx - Reverse proxy et serveur web
- Let's Encrypt - Certificats SSL

# About PawStation Overdrive

## Welcome to PawStation Overdrive!

PawStation Overdrive is an arcade game collection with a "Space Groove" theme. Dive into a cosmic disco universe where reflexes and adrenaline blend with neon lights and electronic music.

## Technologies Used

**Frontend:**
- React 19 - Modern UI framework
- Phaser 3.90 - HTML5 game engine
- Tailwind CSS - Design system
- React Router - Navigation
- Lucide React - Icons

**Backend:**
- FastAPI - High-performance Python framework
- MongoDB - NoSQL database
- Pydantic - Data validation
- Motor - Async MongoDB driver

**DevOps:**
- Docker & Docker Compose - Containerization
- Nginx - Reverse proxy and web server
- Let's Encrypt - SSL certificates
